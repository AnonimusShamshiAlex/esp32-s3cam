## 🤖 СОЗДАНИЕ TELEGRAM БОТА (ПОШАГОВО)

### ШАГ 1: Найти BotFather
1. Откройте **Telegram** на телефоне или компьютере
2. В поле поиска (сверху) введите: **@BotFather**
3. Нажмите на найденного бота (официальный, с синей галочкой ✅)

### ШАГ 2: Создать нового бота
Отправьте BotFather команду (просто введите в поле ввода):

```
/newbot
```

BotFather ответит:
```
Alright, a new bot. How are we going to call it?
Please choose a name for your bot.
```

### ШАГ 3: Ввести ИМЯ бота
Введите любое имя (для людей), например:
```
ESP32 Security Camera
```
Или:
```
Мой охранный бот
```

**Что можно писать:** Любые буквы, цифры, пробелы, эмодзи (но не специальные символы типа @#$%)

### ШАГ 4: Ввести ЮЗЕРНЕЙМ бота
BotFather попросит:
```
Good. Now let's choose a username for your bot.
It must end in `bot`. Like this, for example: TetrisBot or tetris_bot.
```

Введите уникальное имя, которое **ОБЯЗАТЕЛЬНО заканчивается на `bot`**:
```
esp32_security_bot
```
Или:
```
my_camera_2024_bot
```
Или:
```
esp32_s3_cam_bot
```

**Важно!** Если имя занято, BotFather скажет "Sorry, this username is already taken". Придумайте другое.

### ШАГ 5: ПОЛУЧИТЬ ТОКЕН
После успешного создания BotFather напишет:

```
Done! Congratulations on your new bot.

Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

**Этот токен - ваш пароль к боту!** 
- 🔴 НИКОМУ НЕ ПОКАЗЫВАЙТЕ
- 🔴 НЕ ВЫКЛАДЫВАЙТЕ В ИНТЕРНЕТ
- 📋 Скопируйте и сохраните в блокнот

### ШАГ 6: Найти ваш CHAT ID

**Способ 1 (простой):**
1. Найдите своего бота в Telegram (по юзернейму)
2. Отправьте ему любое сообщение: "Привет!"
3. Откройте в браузере ссылку (ЗАМЕНИТЕ на ваш токен):
   ```
   https://api.telegram.org/bot1234567890:ABCdefGHIjklMNOpqrsTUVwxyz/getUpdates
   ```
4. В ответе найдите строчку:
   ```json
   "chat":{"id":123456789}
   ```
   Число `123456789` - это ваш **CHAT_ID**

**Способ 2 (альтернативный):**
1. Напишите боту **@userinfobot**
2. Отправьте ему команду: `/start`
3. Бот покажет ваш ID:
   ```
   Your user ID: 123456789
   ```
   Это и есть CHAT_ID

**ВНИМАНИЕ!** 
- Если вы используете группу, CHAT_ID будет отрицательным: `-123456789`
- Для личного чата CHAT_ID всегда положительный

## 📶 НАСТРОЙКА WI-FI (ДВА ВАРИАНТА)

В коде есть ДВЕ Wi-Fi сети, которые работают одновременно:

### ВАРИАНТ 1: Подключение к домашнему Wi-Fi (основной)

```cpp
const char* WIFI_SSID  = "ESP_Test";        // ← ИЗМЕНИТЕ НА ВАШ
const char* WIFI_PASS  = "1234567890";      // ← ИЗМЕНИТЕ НА ВАШ
```

**Что сюда писать:**
- `WIFI_SSID` - название вашей домашней Wi-Fi сети (как видно на телефоне)
- `WIFI_PASS` - пароль от вашего Wi-Fi

**Примеры:**
```cpp
const char* WIFI_SSID  = "MyHomeWiFi";      // Если сеть называется "MyHomeWiFi"
const char* WIFI_PASS  = "password123";     // Пароль "password123"

// ИЛИ для 5GHz роутеров:
const char* WIFI_SSID  = "MyWiFi_5G";
const char* WIFI_PASS  = "qwerty2024";
```

**Важно!** ESP32 работает только на частоте 2.4GHz. Если у вас роутер с одной сетью 5GHz - переключите его в режим смешанный (2.4+5GHz) или создайте отдельную 2.4GHz сеть.

### ВАРИАНТ 2: Точка доступа ESP32 (всегда включена)

Даже если ESP32 не может подключиться к вашему Wi-Fi, она создает свою сеть:

```cpp
WiFi.softAP("ESP32CAM", "12345678", 6, 0, 4);
//            ↑ИМЯ       ↑ПАРОЛЬ
```

**Что можно изменить:**
```cpp
WiFi.softAP("MyCamera", "88888888", 6, 0, 4);
//            ↑ИМЯ       ↑ПАРОЛЬ
```

**Параметры точки доступа:**
- `"ESP32CAM"` - имя сети (видно на телефоне)
- `"12345678"` - пароль (минимум 8 символов)
- `6` - канал Wi-Fi (1-13, можно не менять)
- `0` - скрытость (0 - видимая, 1 - скрытая)
- `4` - количество подключений (макс 4 устройства)

**Зачем нужна точка доступа:**
- Для настройки ESP32 без Wi-Fi
- Для прямого подключения к камере
- Для отладки

## ⚙️ НАСТРОЙКА КОДА ARDUINO

### 1. Скачайте и установите Arduino IDE
- Ссылка: https://www.arduino.cc/en/software
- Выберите версию для вашей ОС (Windows/Mac/Linux)

### 2. Установите поддержку ESP32
1. Откройте Arduino IDE
2. **Файл → Настройки** (Windows) или **Arduino → Preferences** (Mac)
3. В поле **"Дополнительные ссылки для менеджера плат"** вставьте:
   ```
   https://espressif.github.io/arduino-esp32/package_esp32_index.json
   ```
4. Нажмите **ОК**
5. **Инструменты → Плата → Менеджер плат**
6. В поиске введите **ESP32**
7. Найдите **esp32 by Espressif Systems** и нажмите **Установить** (версия 2.0.14 или новее)

### 3. Установите библиотеку для камеры
1. **Скетч → Подключить библиотеку → Менеджер библиотек**
2. В поиске введите: **esp32-camera**
3. Найдите **esp32-camera by Espressif Systems**
4. Нажмите **Установить**

### 4. ВСТАВЬТЕ ВАШИ ДАННЫЕ В КОД

Откройте код и найдите эту секцию:

```cpp
// ════════════════════════════════════════════
const char* WIFI_SSID  = "ESP_Test";        // ← СЮДА ВАШ Wi-Fi
const char* WIFI_PASS  = "1234567890";      // ← СЮДА ПАРОЛЬ
const char* BOT_TOKEN  = "TG_TOKEN_BOT";    // ← СЮДА ТОКЕН ОТ BOTFATHER
const char* CHAT_ID    = "TG_YOU_ID";       // ← СЮДА ВАШ ID
// ════════════════════════════════════════════
```

**Пример заполнения:**
```cpp
const char* WIFI_SSID  = "MyHomeRouter";
const char* WIFI_PASS  = "myStrongPassword123";
const char* BOT_TOKEN  = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz";
const char* CHAT_ID    = "987654321";
```

**Где взять токен:**
- Откройте диалог с **@BotFather**
- Найдите сообщение с токеном (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)
- Скопируйте ВЕСЬ токен целиком

**Где взять CHAT_ID:**
- Используйте **@userinfobot** (напишите `/start`)
- Или через ссылку с токеном (см. выше)

### 5. НАСТРОЙКА ИНТЕРВАЛА

Найдите в коде:
```cpp
const unsigned long INTERVAL = 5000;  // 5000 мс = 5 секунд
```

**Что можно изменить:**
```cpp
const unsigned long INTERVAL = 30000;   // 30 секунд
const unsigned long INTERVAL = 60000;   // 1 минута
const unsigned long INTERVAL = 300000;  // 5 минут
const unsigned long INTERVAL = 3600000; // 1 час
```

**Совет:** Для тестирования ставьте 5000-10000 (5-10 сек). Для постоянной работы - 60000+ (1 минута и больше), чтобы не перегружать Telegram и не тратить трафик.

## 📥 ЗАГРУЗКА ПРОШИВКИ

### ШАГ 1: Подключите ESP32-S3-CAM
1. Подключите ESP32 к компьютеру через USB-C кабель
2. На плате загорится красный светодиод (питание)

### ШАГ 2: Выберите плату в Arduino IDE
1. **Инструменты → Плата → ESP32 Arduino → ESP32S3 Dev Module**

### ШАГ 3: Настройте параметры платы
**Инструменты →** установите:

| Параметр | Значение |
|----------|----------|
| USB CDC On Boot | **Enabled** |
| USB DFU On Boot | **Disabled** |
| CPU Frequency | **240MHz (WiFi)** |
| Flash Mode | **QIO 80MHz** |
| Flash Size | **8MB (64Mb)** |
| Partition Scheme | **Huge APP (3MB No OTA/1MB SPIFFS)** |
| PSRAM | **OPI PSRAM** |

**Почему именно так:**
- `USB CDC On Boot Enabled` - чтобы работал Serial (вывод в монитор порта)
- `PSRAM OPI PSRAM` - включает дополнительную память для камеры
- `Partition Scheme Huge APP` - больше места под прошивку

### ШАГ 4: Выберите порт
**Инструменты → Порт** → выберите ваш COM-порт:
- **Windows:** COM3, COM4 и т.д.
- **Mac:** /dev/cu.usbmodem*
- **Linux:** /dev/ttyUSB0

Если порта нет:
- Проверьте кабель (должен быть data, не только зарядка)
- Установите драйвер CP210x (для Windows)
- Перезагрузите Arduino IDE

### ШАГ 5: Загрузите прошивку
1. Нажмите **стрелку вправо** (Загрузить) или **Ctrl+U**
2. Дождитесь окончания загрузки (1-2 минуты)
3. Внизу появится: **"Done uploading"**

**Если ошибка загрузки:**
- ESP32-S3 нужно перевести в режим загрузки:
  - Зажмите кнопку **BOOT** на плате
  - Нажмите и отпустите кнопку **RESET**
  - Отпустите кнопку **BOOT**
  - Теперь загружайте прошивку

## 🔧 УСТРАНЕНИЕ ПРОБЛЕМ

### ❌ "❌ Камера: 0x20001" или "❌ Кадр не получен"
**Причина:** Камера не инициализируется
**Решение:**
- Отключите и подключите шлейф камеры (он хрупкий!)
- Проверьте, что шлейф вставлен контактами вниз (к плате)
- Перезагрузите ESP32

### ❌ "⚠️ WiFi нет, статус: 6"
**Причина:** Не подключается к Wi-Fi
**Решение:**
- Проверьте SSID и пароль (регистр букв важен!)
- Убедитесь, что роутер на 2.4GHz
- Перезагрузите роутер
- Временно отключите защиту роутера для теста

### ❌ "❌ Нет соединения с Telegram"
**Причина:** Нет интернета или блокировка
**Решение:**
- Проверьте, что ESP32 получила IP (видно в мониторе)
- Попробуйте отправить фото вручную через браузер:
  ```
  https://api.telegram.org/bot[ТОКЕН]/sendMessage?chat_id=[CHAT_ID]&text=test
  ```
- В России может быть блокировка Telegram API - используйте VPN на роутере

### ❌ "❌ Ошибка: 404" или "❌ Ошибка: 401"
**Причина:** Неправильный токен или CHAT_ID
**Решение:**
- Проверьте, что скопировали ВЕСЬ токен (включая двоеточие)
- Убедитесь, что бот запущен (напишите ему `/start`)
- Проверьте CHAT_ID через @userinfobot

### ❌ ESP32 постоянно перезагружается
**Причина:** Недостаточное питание
**Решение:**
- Подключите к мощному блоку питания (5V 2A)
- Используйте качественный USB кабель
- Отключите светодиод на камере (раскомментируйте код)

### ❌ Фото темное или размытое
**Настройка качества в коде:**
```cpp
cfg.jpeg_quality = 15;    // 0-63, 15 - хорошее качество
cfg.frame_size   = FRAMESIZE_QVGA;  // 320x240
```
Можно изменить на:
```cpp
cfg.jpeg_quality = 10;    // Лучше качество
cfg.frame_size   = FRAMESIZE_VGA;   // 640x480
```
